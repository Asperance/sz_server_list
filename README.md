# Stalzone serverlist

Небольшой неофициальный проект для просмотра актуального списка игровых серверов STALCRAFT по регионам.

Страница помогает быстро найти нужный сервер, посмотреть его адрес и понять, в какой сети и стране он находится. Регионы расположены в удобном порядке: сначала Россия, затем Европа, Америка и Азия.

## Что есть на странице

- актуальный список серверов по регионам и пулам;
- IP-адреса и порты;
- ASN и название сетевого оператора;
- приблизительный город и страна размещения;
- поиск по названию сервера, адресу, оператору или местоположению;
- фильтры по регионам и пулам;
- копирование адресов;
- экспорт списка в JSON и CSV;
- отдельный свёрнутый раздел с историческими адресами, которых сейчас нет в актуальном списке.

Интерфейс адаптирован для компьютеров, ноутбуков, планшетов и телефонов. На больших экранах данные отображаются в таблице, а на мобильных устройствах — в виде удобных карточек.

Список обновляется автоматически в течение дня, поэтому страницу можно использовать как постоянный справочник по доступным игровым адресам.

> Геолокация определяется по IP-адресу и может указывать на сетевую точку присутствия или ближайший город, а не на точное физическое расположение сервера.

Проект создан сообществом и не является официальным сервисом EXBO.

Список серверов обновляется автоматически два раза в сутки — примерно каждые 12 часов.
